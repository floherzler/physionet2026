# Handbook

Updated on: `2026-03-04`

Human-first context that can be discovered on demand by agents.

Use this folder for material that is useful but not required for every implementation task, for example:

- `lorebook/` for physiology/domain background.
- `guild-charter.md` for collaboration norms.
- `map.md` for milestones and roadmap.

Implementation-critical planning stays in `docs/`.

```text
handbook/
```

## How To Use This Folder

- Keep this practical: short entries, concrete decisions, and context that helps future contributors.
- Keep `docs/` as the source of truth for implementation scope and acceptance criteria.
- Use the handbook for mission framing, shared language, and rationale that may not belong in quest docs.
- Primary audience today: core team collaborators in a private GitHub repository.
